# 婚礼照片轮播 - 图片文件夹使用说明

请将您想要在婚礼轮播中展示的照片复制到这个文件夹中。

## 使用说明
1. 照片格式支持：JPG、PNG、GIF等常见图片格式
2. 建议照片尺寸：横屏照片建议宽度在800-1200像素之间，竖屏照片建议高度在800-1200像素之间
3. 照片命名：可以使用任意名称，系统会自动加载所有图片文件
4. 数量限制：建议不超过100张照片，以确保良好的性能

## 注意事项
- 如果image文件夹为空，系统将自动使用默认的示例照片
- 照片会自动适应轮播显示（支持横屏和竖屏照片）
- 系统会按字母顺序加载照片
- 过大的照片可能会影响加载速度，建议适当压缩
